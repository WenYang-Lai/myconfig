# myconfig
Use `curl` or `wget` to install packages and configures!  
Both **linux** and **macos** supported!  
  
**NOTE: myconfig will OVERWRITE your `oh-my-zsh`, `vim` and `tmux` configures!**
  
**via curl**
```shell
bash -c "$(curl -fsSL https://raw.githubusercontent.com/WenYang-Lai/myconfig/master/install.sh)"
```
  
**via wget**  
```shell
bash -c "$(wget https://raw.githubusercontent.com/WenYang-Lai/myconfig/master/install.sh -O -)"
```
